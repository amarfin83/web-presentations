# Values

Status: active presentation area

## Purpose

This folder stores public web presentations for Alexander's Values / Value
Driven AI project.

Each published presentation should live in its own subfolder and include an
`index.html` at that subfolder root.

## Folder Map

- `VDB_AI_Diagnostics_Result/` - planned folder for the VDB AI Diagnostics
  Result web presentation. The folder may be empty locally until the
  presentation files are ready; Git will not track it until it contains at
  least one file.

## Publishing Rules

- Commit only public presentation-ready files.
- Do not commit private project notes, transcripts, raw client/source
  materials, local AI Harness state, credentials or billing data.
- Use stable folder names because the URL path becomes public.
